⚙️ Booking Settings

When the service sub-total is equal to or greater than this amount, 2 resources are required per booking. Below this amount, only 1 resource is required.

XCG
Example: If threshold is 161 XCG:
  • Sub-Total of 160 XCG → 1 resource needed → More slots available
  • Sub-Total of 161 XCG or more → 2 resources needed → Fewer slots available
✅ Default Availability Schedule: Resources are available by default with the following schedule:
  • Monday - Friday: Morning (2 slots), Afternoon (2 slots), Evening (1 slot)
  • Saturday: Morning only (2 slots)
  • Sunday: Closed

Use the 📅 Calendar button on each resource to override specific days if needed (e.g., mark someone as unavailable for vacation).
